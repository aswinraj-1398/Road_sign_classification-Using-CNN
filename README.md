# Road Sign Classification Using CNN

## Project Overview
This project aims to classify road signs using Convolutional Neural Networks (CNN). The goal is to develop a model that can accurately identify different types of road signs to enhance traffic safety and assist autonomous driving systems.

## Features
- Classifies multiple types of road signs.
- Utilizes CNN for image recognition.
- User-friendly interface for testing the model.
- High accuracy with a well-optimized model.

